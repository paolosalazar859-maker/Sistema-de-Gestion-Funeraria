2. Resultado:
   ────────────────────────────────────────────────────────
   dist-cliente/
   ├── AURA-1.0.0-Instalador-Mac.zip       (~80 MB)

3. Enviar al cliente:
   ────────────────────────────────────────────────────────
   • Cliente con Mac → Mac.zip
   • Cliente con Windows → Windows.zip
   • No sabes qué tiene → Completo.zip

  📋 QUÉ SE COMPILA
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


El script preparar-distribucion-completa.sh hace:

1. Compila con Vite
   ────────────────────────────────────────────────────────
   → Optimiza React para producción
   → Minimiza archivos
   → Genera build en dist/


2. Construye instalador Mac
   ────────────────────────────────────────────────────────
   → Crea AURA-1.0.0-mac.dmg
   → Empaqueta con electron-builder
   → Incluye base de datos SQLite


3. Construye instalador Windows
   ────────────────────────────────────────────────────────
   → Crea AURA-Setup-1.0.0.exe
   → Instalador NSIS profesional
   → Incluye base de datos SQLite


4. Organiza en paquetes
   ────────────────────────────────────────────────────────
   → 3 carpetas con instaladores
   → Instrucciones específicas por sistema
   → Todo comprimido en ZIP

Mac:
────────────────────────────────────────────────────────────
• macOS 10.13 (High Sierra) o superior
• macOS 11 (Big Sur) o superior recomendado
• 500 MB de espacio libre
• Procesador Intel o Apple Silicon (M1/M2/M3)


Windows:
────────────────────────────────────────────────────────────
• Windows 10 (64-bit) o superior
• Windows 11 recomendado
• 500 MB de espacio libre
• Procesador x64


Ambos:
────────────────────────────────────────────────────────────
• 4 GB RAM mínimo (8 GB recomendado)
• Pantalla 1280x720 mínimo
• Sin necesidad de internet (funciona offline)

Solo puedes compilar desde Mac:
────────────────────────────────────────────────────────────
⚠️  Para crear el instalador de Mac (.dmg), DEBES estar
    en una Mac.

⚠️  Electron-builder no puede crear DMG desde Windows.

✅  Desde una Mac puedes crear ambos (Mac + Windows)

✅  Si solo tienes Windows, contacta para soluciones


Primera instalación en clientes:
────────────────────────────────────────────────────────────
🍎 Mac: Requiere "click derecho → Abrir" (solo 1ra vez)
🪟 Windows: Puede mostrar advertencia de Windows Defender


Los datos son locales:
────────────────────────────────────────────────────────────
✓ No se sincronizan automáticamente entre computadoras
✓ Cada instalación tiene su propia base de datos
✓ Para compartir datos, usa exportar/importar
